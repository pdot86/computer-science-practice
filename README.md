# computer-science-practice

This is a repo that conatains information, common data structures, algorithms, and interview style problems. I put this repo together to help refresh, learn, and study for interviews. Included is answers to specific questions from common Data Structures and Algorithms practice sites like [LeetCode](./src/leetcode/).

## Setup

This project uses the [Bun](https://bun.sh/) runtime to execute the [TypeScript](https://www.typescriptlang.org/) files directly.

### To install dependencies:

```bash
bun install
```

### To run:

```bash
bun run index.ts
```
